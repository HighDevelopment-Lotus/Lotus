Mijn Objecten

Weet je zeker dat je het object wilt kopen?

  • WASD/Muis om camera te besturen.
  • Pijltjestoetsen om Object te bewegen.
  • PAGE UP/DOWN om Object omhoog/omlaag te bewegen.
  • LEFTALT om Object op de grond te plaatsen.
  • F3 om te switchen tussen roteren/bewegen.
  • ENTER om object te kopen/plaatsen.
  • F5 om muis aan/uit te schakelen.
  • ESC om Decoratie modus uit te schakelen.
