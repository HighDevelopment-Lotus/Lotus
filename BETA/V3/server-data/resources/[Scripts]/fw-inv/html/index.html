Keybinds (Standaard):

  • Ctrl + C: Kopieert de item informatie waar je op hovert
  • Shift + Rechter Muis: Quick move de helft van de items.
  • Ctrl + Rechter Muis: Quick move één item.
  • Linker Muis: Item slepen.
  • Rechter Muis: Quick move.
  • Middel Muis: Item gebruiken.
